# Console Games
<img src="img/main_menu.gif"></img>

An application to be run by a command line and it contains 2 command line games.<br><br>
### How to build: (Building requires Windows)
1. Install Makefile from MinGW or Chocolatey.<br>
2. Install GCC from MinGW or TDM-GCC.<br>
3. Run the batch file to build.<br>
   It should take about 5 seconds.<br>
4. Open the compiled EXE (if it didn't open)<br>

### What I used:<br>
1. Requirements: Makefile and GCC<br>
2. My IDE is Dev-C++<br>

# Games
<h2>Guessing Game</h2>
Difficulties:<br>
- Easy: Random number between 1 and 50<br>
- Normal: Random number between 1 and 100<br>
- Hard: Random number between 1 and 250<br>
- Insane: Random number between 1 and 1000<br>
